# javascript syntax

## 변수와 상수

- let: 변수 만들 때 사용
- const: 상수 만들 때 사용
- var: 안쓰임

## 자료형

- 숫자형
- 문자형
- 불린형
- null 값만을 위한 독립 자료형입니다. null은 알 수 없는 값을 나타냅니다.
- undefined 값만을 위한 독립 자료형입니다. undefined는 할당되지 않은 값을 나타냅니다.
- 객체형 복잡한 데이터 구조를 표현할 때 사용합니다.
- 심볼형 객체의 고유 식별자를 만들 때 사용합니다.
